Req. Steam, Among us (on steam),BepInEx 86x, MalumMenu.
Open steam, browse among us's local files,
download BepInEx 86x from the official github
open the .rar file and copy all the contents inside
go to your among us folder and paste BepInEx in the same folder where Among Us.exe is located
once you do that, open among us, then close it out. (It creates the BepInEx files)
if there isn't already, make a plugins folder (Among Us->BepInEx->plugins)
then put MalumMenu inside. Open among us and Voila. To open, press 'Del' on the keyboard. (Can be changed in the config)
